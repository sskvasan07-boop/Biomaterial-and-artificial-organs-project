🧵 Silk Biomaterial Tensile Testing Simulator📌 Project OverviewThe Silk Biomaterial Tensile Testing Simulator is an interactive digital twin environment designed to analyze the mechanical properties of Silk Fibroin (Bombyx mori). This tool allows biomedical researchers and students to simulate tensile stress tests on silk fibers, comparing the performance of Degummed vs. Non-degummed silk without the need for expensive physical testing equipment.By adjusting geometry and applied loads, users can visualize the stress-strain relationship and determine the suitability of silk for specific tissue engineering scaffolds (Vascular, Bone, Neural, or Ocular).✨ Key FeaturesDynamic Tensile Simulation: Real-time calculation of Stress ($\sigma$), Strain ($\epsilon$), and Elastic Modulus ($E$).Material Comparison: Toggle between degummed and non-degummed states to see how sericin removal affects mechanical toughness.Interactive Stress-Strain Curve: Visual tracking of the "Operating Point" relative to the Yield Point and Ultimate Tensile Strength (UTS).Biocompatibility Radar: A multi-axis comparison of material properties including processability and cell-adhesion potential.Physics Engine: Built-in reference for Hooke's Law and cross-sectional area calculations for cylindrical specimens.Report Generation: Capability to download simulation data for further laboratory analysis.🛠️ Tech StackLanguage: PythonFrontend/Simulation Engine: Antigravity / StreamlitData Analysis: NumPy, PandasVisualization: Plotly / Matplotlib (for the Stress-Strain Curve and Radar charts)🚀 Getting StartedPrerequisitesEnsure you have Python 3.8+ installed on your system.InstallationClone the repository:Bashgit clone https://github.com/your-username/silk-tensile-simulator.git

📊 Methodology & PhysicsThe simulator utilizes standard engineering formulas to ensure scientific accuracy:Engineering Stress: $\sigma = \frac{F}{A_0}$Engineering Strain: $\epsilon = \frac{\Delta L}{L_0}$Elastic Modulus: $E = \frac{\sigma}{\epsilon}$The UTS (Ultimate Tensile Strength) for the Bombyx mori fibroin in this simulation is set at 650 MPa, reflecting high-grade degummed silk standards.🏥 Biomedical ApplicationsThis project focuses on four primary clinical domains:Vascular Grafts: Small-diameter vessel simulation.Bone Tissue Engineering: Mineralized silk scaffolds.Neural Guidance: Nerve regeneration conduits.Ocular Implants: Transparent thin-film characterization.🤝 Contributors[Your Name] - Lead Developer & Biomedical Researcher[Team Member Name 1][Team Member Name 2]📄 LicenseThis project is licensed under the MIT License - see the LICENSE file for details.
